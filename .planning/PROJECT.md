# GuardianAI — AQX Sports Analytics Hackathon

**Type:** Hackathon Project (AQX Sports Analytics)
**Deadline:** Jun 26, 2026 @ 12:30pm GMT+5:30 (~14 hours)
**Tech Stack:** Python 3.10+, Streamlit, XGBoost, SHAP, Plotly, Pandas
**Sport:** NBA (Basketball)

## Vision

GuardianAI is an injury risk and load management dashboard that gives coaches a real-time "who to rest tonight" recommendation using player workload data, rolling averages, and the Acute:Chronic Workload Ratio (ACWR) — a proven sports-science metric. Powered by XGBoost + SHAP explainability, it surfaces plain-English insights rather than raw numbers.

## Problem

NBA teams lose tens of millions annually to preventable injuries. Most load-management decisions are based on gut feel. GuardianAI provides a data-driven, explainable alternative.

## Core Features

1. Player Risk Cards (Red/Yellow/Green) with injury probability %
2. SHAP-based explanations ("why is this player flagged?")
3. Team overview scatter plot (workload vs. risk)
4. Model performance metrics (AUC, precision, recall)
5. Team/date selector dropdown
6. Dark-theme Streamlit dashboard
